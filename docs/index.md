# Documentation

This directory contains detailed guides for using `kpi-collector`.

## Guides

- [Getting Started](getting-started.md): install, collect your first metrics, and verify results in 5 minutes.
- [Installation](installation.md): install, verify, and uninstall the CLI.
- [Collecting Metrics](collecting-metrics.md): authentication modes (kubeconfig vs manual), behind-the-scenes details, manual `oc` steps, and CPU placeholders.
- [KPI Configuration](kpis-file-configuration.md): KPI file format, sampling frequency/duration, run modes, range queries, and KPI profiles.
- [Database Commands](database-commands.md): query and manage stored data with `db show` and `db remove`.
- [Grafana](grafana.md): start/stop Grafana, datasource setup, and dashboard usage.
- [Troubleshooting](troubleshooting.md): common issues and solutions.
- [AI Skill](ai-skill/README.md): use the AI agent skill to generate KPI files and run the tool from your IDE.
